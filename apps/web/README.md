# streamui

Open source streaming UI components for AI applications. Built on top of shadcn/ui and AI SDK 6.

## Documentation

Visit [streamui.dev](https://streamui.dev) to view the documentation.

## Features

- **Progressive rendering**: Components update smoothly as data streams in
- **Schema-first**: Zod schemas that work with AI SDK's `Output.object()`
- **AI SDK 6 compatible**: Built for structured streaming with `streamText`
- **shadcn compatible**: Install via the shadcn CLI
- **Type-safe**: Full TypeScript support

## Quick Start

```bash
npx shadcn@latest add "https://streamui.dev/r/weather-card.json"
```
